# pip

## 安装

```bash
pip install <package_name>
```



## 查看

当前环境安装的列表

```bash
pip list
```

查看某个特定的包是否安装

```bash
pip show <package_name>
```

将当前环境安装的包输出到 `requirements.txt` 文件中

```bash
pip freeze > requirements.txt
```



## 卸载

```bash
pip uninstall <package_name>
```
